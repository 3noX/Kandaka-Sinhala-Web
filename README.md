# 🔥 කඳක | Kandaka

**කඳක** is a fun, creative Sinhala-language website project developed purely for educational and experimental purposes. The site uses a dark theme and glowing red aesthetics to represent a bold and energetic cultural tone.

> ✨ Designed and developed by [Chathura Denuwan]([https://www.linkedin.com/in/chathura-denuwan](https://github.com/3noX))

## 📌 Features

- 🌑 Dark-themed Sinhala interface
- 🔥 Neon red visual style with custom typography
- 📷 Achievements section with a modern photo gallery layout
- 📨 Contact form (UI only)
- 🎨 Sinhala headings and navigation
- ⚙️ Responsive UI elements (basic)

---

## 🎯 Purpose

This project was built as a fun creative idea for learning web development and expressing Sri Lankan identity through code. It is **not meant for commercial use** — only for **educational** and **portfolio** purposes.
